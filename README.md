This is a production-ready starter for **AutoConnectorsApp** with a flashy UI (TailwindCSS + Framer Motion).

## What you get
- Next.js frontend (App Router)
- TailwindCSS for styling & animations
- Framer Motion for smooth UI animations
- Firebase Auth + Firestore skeleton
- Stripe placeholders for Connect
- Cloud Functions skeleton
- Admin dashboard, Connector dashboard, Lead flow, Commission logic

## How to deploy (quick)
1. Unzip and open folder on your computer (or upload to GitHub from iPad).
2. Copy `.env.local.example` -> `.env.local` and fill values (Firebase + Stripe).
3. Install deps: `npm install`
4. Run dev: `npm run dev` and open http://localhost:3000
5. Deploy: push to GitHub and use Vercel to import the repo, set environment variables in Vercel, then deploy.

Detailed step-by-step deploy instructions in DEPLOY.md inside this repo.
